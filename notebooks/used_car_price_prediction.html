Modeling Phase¶

The correct order in modeling phase is:

  • Define X and y
  • Detect numerical/categorical columns
  • Split into train/test
  • Build the pipeline with those lists
  • Fit on the training set
  • Evaluate on the test set

Model Performance Comparison Graph¶

After comparing multiple regression models, XGBoost with Early Stopping achieved the best overall performance, obtaining the lowest prediction errors and the highest R² score. This confirms that gradient boosting methods are highly effective for structured tabular datasets such as used car pricing problems.

The project also demonstrated the importance of feature engineering, especially variables such as car_age, km_per_year, and brand extraction.

Final Conclusions¶

This project developed a machine learning system to predict used car prices using real market variables such as brand, mileage, age, fuel type, transmission, and seller information. Several regression models were evaluated, including Linear Regression, Random Forest, XGBoost, and CatBoost. Among all tested approaches, XGBoost with *Early Stopping achieved the best overall performance, obtaining the lowest prediction errors and the highest explanatory power (R² = 0.89). Feature importance analysis revealed that transmission type, premium brands, fuel type, and vehicle age were among the most influential factors affecting price prediction. The project demonstrates the importance of data preprocessing, feature engineering, and model selection when solving structured tabular machine learning problems.*
